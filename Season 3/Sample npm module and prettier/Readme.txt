----------------------------------command-------------------------------------
install prettier =>                     npm install --save-dev prettier
make config file for prettier =>        echo {}> .prettierrc.json
install sync eslint & prettier =>       npm install --save-dev eslint-config-prettier
check project with prettier =>          npx prettier --check .
check file with prettier =>             npx prettier --check ***.js(your file name)
fix project with prettier =>            npx prettier --write .
fix file with prettier =>               npx prettier --write ***.js(your file name)
